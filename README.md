# WPF Repository App

## What the Project Does

The WPF Repository App is a desktop application built using the Windows Presentation Foundation (WPF) framework. This application allows users to manage their GitHub repositories conveniently through a beautiful and user-friendly interface. Users can create, view, and delete projects, each containing essential details such as the project name, GitHub link, and description. The app ensures that users can access and manage only their own projects, providing a personalized experience.

## Why the Project is Useful

- **Centralized Repository Management:** Easily keep track of your GitHub repositories in one place.
- **User-Friendly Interface:** Enjoy a visually appealing and intuitive design, making repository management a pleasant experience.
- **Personalized Experience:** Access only your own projects, ensuring that your data is private and secure.
- **Efficiency:** Quickly add new repositories, view existing ones, and delete those you no longer need.

## How Users Can Get Started with the Project

1. **Download the Application:**
   [Download the latest release]() from the site.

2. **Clone the Repository (Optional for Developers):**
   ```sh
   git clone https://github.com/SashaNikitich/WPF_RepositoryApp.git

3. **Navigate to the Project Directory (Optional for Developers):**
   ```sh
   cd WPF_RepositoryApp

4. **Open the Solution in Visual Studio (Optional for Developers):**
   Open `WPF_RepositoryApp.sln` in Visual Studio.

5. **Build the Solution (Optional for Developers):**
   Build the solution by selecting `Build > Build Solution` from the menu or by pressing `Ctrl+Shift+B`.

6. **Run the Application:**
   Start the application by executing the downloaded file or by selecting `Debug > Start Debugging` or by pressing `F5` in Visual Studio.
